# 📘 Problem Statements – Week 03 & 04

## 📌 Overview

This repository contains solutions to **sorting and searching problems** implemented in Java as part of Week 03 & 04 coursework.

The project demonstrates the use of different algorithms based on data size, efficiency, and problem requirements.

---

## 🧠 Concepts Covered

* Bubble Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Linear Search
* Binary Search
* Time Complexity Analysis
* Stability in Sorting

## 🔹 Problem Descriptions

### ✅ Problem 1: Transaction Fee Sorting

* Bubble Sort for small datasets
* Insertion Sort for medium datasets
* Stable sorting and high-fee detection

### ✅ Problem 2: Client Risk Ranking

* Sorting risk scores using Bubble & Insertion Sort
* Identifying top high-risk clients

### ✅ Problem 3: Trade Volume Analysis

* Merge Sort for stable sorting
* Quick Sort for descending order
* Merging sorted datasets

### ✅ Problem 4: Portfolio Return Sorting

* Sorting based on return rate and volatility
* Use of hybrid sorting techniques

### ✅ Problem 5: Account ID Lookup

* Linear Search for unsorted data
* Binary Search for efficient lookup in sorted data

### ✅ Problem 6: Risk Threshold Lookup

* Binary Search to find floor and ceiling values
* Efficient threshold analysis

## 📊 Time Complexity Summary

| Algorithm      | Best Case  | Average Case | Worst Case |
| -------------- | ---------- | ------------ | ---------- |
| Bubble Sort    | O(n)       | O(n²)        | O(n²)      |
| Insertion Sort | O(n)       | O(n²)        | O(n²)      |
| Merge Sort     | O(n log n) | O(n log n)   | O(n log n) |
| Quick Sort     | O(n log n) | O(n log n)   | O(n²)      |
| Linear Search  | O(1)       | O(n)         | O(n)       |
| Binary Search  | O(1)       | O(log n)     | O(log n)   |
